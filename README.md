# Jogo da Velha (Tic Tac Toe)

Projeto desenvolvido para a disciplina de Linguagens de Programação.

## Integrantes do Grupo

- Guilherme Mendes
- Lucas Simioni
- Matheus Quaranta

## Objetivo

Desenvolver um jogo da velha utilizando HTML, CSS e a linguagem de programação JavaScript, aplicando conceitos de desenvolvimento web, lógica de programação, manipulação do DOM e controle de eventos.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Funcionalidades

- Criação dinâmica do tabuleiro
- Alternância entre jogadores X e O
- Destaque visual da casa selecionada
- Verificação automática de vitória
- Verificação automática de empate
- Reinício automático após o término da partida
